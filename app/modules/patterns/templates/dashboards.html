Patterns

Dashboards

  • Information Dashboards
  • Workflow Entry Points

When designing dashboards think about which information and functions will be highest priority to the user and then organize your layout accordingly.

Information Dashboards

Simple Cards (Menu Open)

Use simple cards when you need to show multiple streams of information. These can be related or unrelated, but should share the same level of user importance to justify being included in the dashboard.

Simple Cards (Menu Closed)

Complex Data

A complex data dashboard allows you to isolate important data and show it in combination with other highly relevant information.

Workflow Entry Points

Action Launcher

Use the action launcher when the number of actions available to the user is minimal and you have only have a few pieces of non-field-related data to display.

Task Gallery (Grid)

The task gallery is primarily used as a control panel for products that provide many possible functions or actions at launch.
